# 原理图
## 目标
- 学会读懂原理图，芯片和外接的芯片引脚如何对接？
- 弄清楚接口的协议，数据怎么传输？引脚怎么配合？
- 设置时序，芯片发出信号，符合外接芯片的要求
	- 看芯片手册能设置哪些参数，弄清楚含义（以CLK为单位）
	- 看外设芯片手册，弄清楚参数的取值范围（以秒为单位）
	- 计算芯片寄存器取值
